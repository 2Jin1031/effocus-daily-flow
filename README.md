## 페르소나 및 사용자 시나리오

### 페르소나: 바쁜 우테코 크루, 칼리(2N세)
- 월: 13시 출근 / 화~금: 10시 출근 / 주말: 유동적
- 정신 없이 다양한 메모가 생기며 정리 시간 부족
- 목표: 최대한 효율적으로 시간 사용, 반복 작업 자동화 희망

### 사용자 시나리오
1. 어플 실행 → 오늘 일정 기반으로 자동 경로 추천
2. 예: "8시 23분 출발 시 회의 5분 전 도착"
3. 빠르게 준비 후 출발 결정

### 인수 조건
- Given: 사용자가 앱에 접속했을 때  
- When: 특정 시각이 되었을 때  
- Then: 현재 해야 할 일을 자동 안내


## ** 사용자 시나리오: 바쁜 우테코 크루 칼리**

**목표:** 오늘의 우테코 일정을 놓치지 않고, TODO를 빠뜨리지 않으며 최적의 시간에 이동 및 실행하기

---

#### **시나리오:**
  
우테코 크루 칼리는 오늘도 정신없이 하루를 시작한다.

오전 9시 10분, 칼리는 분주히 준비하면서 Effocus 앱을 실행한다.

1. 앱 메인 화면에는 오늘의 일정이 자동으로 구성되어 있다.
    
    - 10:00 데일리 미팅
        
    - 10:30 레벨3 오리엔테이션 강의
        
    - 12:00 점심
        
    - 16:00 다이소 (칫솔 구매)
        
    - 18:00 헬스장
        
    
2. 앱은 현재 위치(주거지)에서 루터회관까지의 이동 시간을 계산하여 칼리에게 알려준다.
    
    - “지금 출발하면 9:54 도착, 6분 여유 있음.”
        
    - “8시 23분 출발 시 회의 5분 전 도착 예상”
        
    
3. 칼리는 알림을 받고 준비를 마친 뒤 출발을 결정한다.
    
4. 오전 일정이 끝나고 점심시간 직전, 앱이 자동으로 칼리의 TODO를 리마인드한다.
    
    - “지금 다이소가 경로 근처입니다. ‘칫솔 구매’ TODO 실행하시겠어요?”
        
    - 수락 시 경로에 자동 추가됨.
        
    
5. 헬스장 가는 길에 앱은 다시 여유 시간과 TODO를 기반으로 루트를 설계하여 안내한다.
    
    - “지금 출발하면 도착 5분 전 도착 가능. 여유 시간 12분 있음.”
        
    
6. 하루가 끝난 후 앱은 오늘의 일정 완료율, 이동 거리, TODO 완료 상태를 자동 시각화해서 보여준다.
    
    - 완료율 87%, 이동 거리 6.3km

### **사용자 스토리와 인수 조건 예시**

  

**요구사항:** 위치 기반 TODO 리마인드 및 자동 경로 반영 기능

  

**사용자 스토리:**

“우테코 크루로서, 저는 특정 장소 근처에 도달했을 때 관련 TODO를 자동 안내받고 싶습니다. 그래서 경로를 놓치지 않고 효율적으로 일정을 소화할 수 있습니다.”

  

**인수 조건:**

- **Given**: 사용자가 앱에 로그인했고, TODO에 ‘다이소 - 칫솔’이 등록되어 있음
    
- **When**: 사용자가 다이소 반경 100m 이내에 진입함
    
- **Then**: 앱은 ‘다이소 - 칫솔’ TODO를 자동 알림으로 표시하고, 수락 시 경로에 추가함

## 핵심 가치 제안

1. **인지 부담 최소화**  
   - 일상 패턴 학습 → 자동 제안  
2. **즉각 결정 지원**  
   - 사용자는 결정만 함 → 결정 피로 해소  
3. **실행 중심 구조화**  
   - 캘린더, 지도, TODO 없이 실행에 집중  
4. **행동 변화 유도**  
   - 출발·도착 타이밍, 위치 기반 행동 자동 안내

## 핵심 기능

### 1. 일정 예측 및 출발 유도
- 네이버맵 API 기반 실시간 이동 시간 예측
- 출발 시각 자동 추천

### 2. 장소 기반 TODO
- 메모 기반 장소-행동 추출 (예: "다이소 - 칫솔")
- 해당 장소 도착 시 자동 알림

### 3. 동선 기반 TODO 리마인드
- 현재 경로 기반 TODO 제안
- 제안 수락 시 경로에 자동 반영

### 4. 하루 일정 자동 설계 + 시각화
- 일정, TODO, 위치 기반 타임라인 생성
- 이동 포함 전체 스케줄 시각화
- 여유 시간, 예상 지연, 완료율 표시
